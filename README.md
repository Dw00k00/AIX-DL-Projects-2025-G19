# 딥러닝을 이용한 노래 음성 스타일 전이(singing-voice-conversion)
Singing Voice Conversion (SVC) Project  
딥러닝 기반 노래 음성 스타일 전이 프로젝트

---

##  Overview  
이 프로젝트는 **딥러닝 기반 Singing Voice Conversion(SVC)** 기술을 활용하여  
“멜로디·가사·리듬은 그대로 유지하고, 음색만 특정 가수 스타일로 변환하는”  
end-to-end 오디오 변환 시스템을 구축하고 분석하는 것을 목표로 한다.

**So-VITS-SVC / RVC** 등 공개된 오픈소스 모델을 기반으로  
전처리 → 특징 추출 → 모델 학습 → 추론 → 평가  
전 전체 파이프라인을 실험했다.
